# AI Dataset Meta Collector

AI Dataset Meta Collector는 폴더 내 데이터 파일의 메타데이터를 자동 수집하고 분석하여  
데이터셋 구조를 파악하고 관리할 수 있도록 지원하는 데스크톱 기반 데이터 분석 도구입니다.

대량의 데이터 파일을 대상으로 메타데이터 수집, 데이터셋 요약 생성, 시각화 및 검색 기능을 제공하여  
데이터 자산 관리 및 AI 데이터셋 분석에 활용할 수 있습니다.

---

# 주요 기능

## 파일 탐색 및 관리
- 폴더 단위 파일 스캔
- 파일 목록 테이블 표시
- 파일 선택 기반 메타데이터 조회

## 메타데이터 수집
- 파일 기본 메타데이터 자동 수집
- Windows Explorer 기반 파일 속성 수집
- 파일 해시 기반 중복 파일 탐지

## 파일 유형별 분석
- CSV 데이터 구조 분석
- Excel 파일 정보 분석
- JSON 구조 분석
- 이미지 파일 정보 분석
- PDF 문서 정보 분석

## 메타데이터 저장
- SQLite 기반 메타데이터 인덱스 저장
- 파일 이름 기반 메타데이터 검색 기능 제공

## 데이터셋 분석
- 전체 데이터셋 요약 생성
- 파일 유형 분포 분석
- 데이터셋 크기 및 통계 계산

## 시각화
- 데이터셋 파일 유형 분포 그래프 생성

## 파일 미리보기
- 이미지 파일 미리보기
- 텍스트 기반 파일 내용 확인

## 데이터 내보내기
- 개별 파일 메타데이터 다운로드
- 전체 메타데이터 ZIP 다운로드
- 메타데이터 CSV 다운로드

## 웹 대시보드
- Flask 기반 데이터셋 요약 API 제공

## 성능 개선
- 멀티스레드 기반 메타데이터 수집
